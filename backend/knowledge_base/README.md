# База знаний ПСБ

База знаний разделена по категориям для удобства управления и обновления.

## Структура файлов:

- **credit.txt** - Кредитование (ипотека, автокредит, потребительский кредит)
- **cards.txt** - Дебетовые и кредитные карты
- **insurance.txt** - Страхование (ОСАГО, КАСКО, страхование жизни)
- **investments.txt** - Инвестиции и накопления
- **online_banking.txt** - Онлайн-банкинг (ПСБ Онлайн)
- **cashback.txt** - Кэшбэк и бонусы
- **accounts.txt** - Открытие/закрытие счетов
- **contacts.txt** - Контакты и отделения

## Как обновлять данные:

1. Откройте соответствующий файл
2. Внесите изменения
3. Сохраните файл
4. RAG система автоматически подхватит изменения при следующем запросе

## Формат данных:

Данные хранятся в текстовом формате. Каждый файл содержит информацию по соответствующей категории услуг ПСБ.
